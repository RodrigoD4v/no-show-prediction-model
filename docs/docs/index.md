# No Show Prediction Model documentation!

## Description

Projeto de ciência de dados voltado à predição de faltas em consultas médicas, utilizando aprendizado de máquina para identificar pacientes com maior probabilidade de não comparecimento. O objetivo é apoiar decisões que reduzam o número de ausências e melhorem a eficiência do atendimento em clínicas e sistemas de saúde.

The Makefile contains the central entry points for common tasks related to this project.
